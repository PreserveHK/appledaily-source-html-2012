港童舞龍蝦　混醬最叻

發脾氣、無禮貌、要人服侍？未必每個小朋友都是可怕的港童！這天他們穿起廚師服拿起鑊鏟，和爸爸媽媽合力炮製連大廚都未必能掌握得好的生猛龍蝦，誓要用道道龍蝦佳餚為香港肯學肯做的小朋友洗去「港童」的污名！　記者：容慧心　攝影：陳盛臣

多溝通煮出好味來

上月尾當暑假還未結束，一班小朋友連同家長參加了一個名為「大『煮』人．小廚神」的親子廚藝比賽，為這個暑假劃上完美的句號。今次的烹飪比賽，主角是龍蝦！生猛龍蝦不是容易處理的食材，大人都未必會煮，更何況是小朋友？「我發現小朋友其實對自己在街外吃過的東西最好奇，我們比賽的主題是健康，於是想到讓他們在超巿內自選蔬菜，烹調平日在家較少煮的新鮮龍蝦，滿足他們的好奇心外，也可考驗家長與子女的合作性。」評判之一的Superlife Culture Club行政總廚Charmaine說。
烹飪技巧不是這次比賽的重點，因現在的小朋友有很多機會參加烹飪班，對入廚絕不陌生，「反而家長經常長時間工作，少與子女溝通，因為合作和味道是掛鈎的，家長和子女溝通得好，做出來的菜自然是預期中的美味。」Charmaine興奮地說。「起初以為小朋友會驚生龍蝦，怎料他們反而很主動去揀靚龍蝦，表現比平日做曲奇更投入！以後的小朋友廚藝班都可考慮選用更多生猛的食材了。」

Charmaine@Superlife Culture Club行政總廚

曾於國際六星級酒店餐廳工作和擔任日本餐廳主廚，現常擔任不同活動的客席廚師，喜愛周遊列國學習各地美食，編寫創意食譜和開設廚藝班分享入廚心得。

鴛鴦蘆笋龍蝦沙律

龍蝦是美味又矜貴的海鮮，越簡單的烹調越能吃出龍蝦的鮮味，李氏父子利用蘆笋和有「王子蔬菜」之稱的白蘆笋，加入自家的秘製醬汁中，做成這道開胃沙律，務求以「less is more」之道迎合大家的胃口！

材料

急凍熟龍蝦肉300克、青蘆笋（刨皮）/白蘆笋（刨皮）各4條、茄汁/沙律醬各50克、喼汁/鹽/黑胡椒碎/橄欖油各少許、檸檬（榨汁）1個

做法

1. 將急凍熟龍蝦解凍，瀝水備用。

2. 煲滾水，下橄欖油和鹽，加入青蘆笋和白蘆笋，烚約3分鐘，撈起放入冰水浸10分鐘。

3. 做醬汁，茄汁、沙律醬、鹽、黑胡椒碎、檸檬汁和喼汁拌勻。

4. 青白蘆笋切段，梅花間竹地排在碟上，灑上黑胡椒碎調味。

5. 龍蝦加入醬汁拌成沙律，放在蘆笋中間，完成。

Dad's Talk

經常與兒子討論甚麼是「好味」，希望他知道利用不同食材做菜，並享受烹調過程，而囝囝從旁幫手，大大增加親子關係！

龍蝦番茄凍湯

羅氏父子估計其他參賽者多數會做龍蝦沙律或香煎原隻龍蝦等菜式，於是想到用番茄做一道凉快清新的凍湯，提起大家的胃口，也貼心地讓評判可以享用一次龍蝦全餐。

材料

龍蝦1隻、番茄4個、青檸（榨汁）2個、香茅2條、鹽/糖/黑胡椒碎各少許

做法

1. 番茄表面十字，放滾水略燙至番茄皮翻起，盛起去皮，橫切去頂，刮去中央的果瓤部份，果瓤及番茄殼分別留用。

2. 果瓤加青檸汁、鹽和糖用攪拌機打成湯，坐冰水浸至少10分鐘。

3. 龍蝦放滾水烚至半熟，撈起，拆肉。

4. 香茅輕拍後，放平底鑊中，上放龍蝦肉，以小火煎熟後放凉。

5. 把凍湯放在番茄殼，加入龍蝦肉，灑上黑胡椒碎即可。

Son's Talk：

比賽前特地到現場視察，本來想做燒龍蝦，卻發現廚具未能配合，故改白烚，事前分配好爸爸煮龍蝦，我整番茄，臨場才沒有大亂陣腳！

舞動龍蝦

生猛的龍蝦手舞足蹈，令悅婷有感而發，想到這個菜名，新鮮的龍蝦加上平日愛吃的蘑菇和香滑忌廉汁，悅婷和媽媽都希望與其他人分享自己喜歡的味道。

材料

龍蝦1隻、乾葱（切碎）5粒、大蘑菇2隻、忌廉奶/白酒各200毫升、橄欖油/羅勒（切碎）/鹽/胡椒粉/法國芥末醬各少許

做法

1. 大蘑菇洗淨切片。

2. 燒熱平底鑊下橄欖油，落乾葱爆香，加蘑菇片略炒，下半份白酒、法國芥末醬、羅勒和忌廉奶，煮至蘑菇軟身成醬汁。

3. 龍蝦切件加橄欖油、鹽和胡椒粉調味後，下鑊用小火煎2分鐘，加餘下白酒煮至龍蝦半熟，盛起，放入醬汁中，將龍蝦肉煮至熟透即可。

Mom's Talk：

想不到法國芥末醬帶點酸味，剛好中和了忌廉汁的肥膩，也提升了龍蝦的鮮味。
