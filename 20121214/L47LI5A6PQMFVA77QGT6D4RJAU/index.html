澳洲小廚神　Greta愛辦上菜

澳洲小廚神Greta Yaxley，兩歲已懂搓麵糰，十歲攞獎！她的律師媽媽和核數師爸爸，是否日夜特訓她成材呢？「我收到大會通知獲面試機會的電郵，才知道她參加比賽了！」聽媽媽說着，Greta發出小朋友奸計得逞的大笑。
記者：容慧心　攝影：伍慶泉　蔡家輝

部份相片由Greta-Junior MasterChef Australia facebook官網提供

Profile

2011年澳洲小廚神冠軍得主，得獎後經常獲邀出任美食節目嘉賓和比賽評判，夢想開餐廳或當烹飪節目主持。

媽媽做軍師

這次由媽媽陪伴來港教港童做聖誕大餐，比賽時也是由媽媽陪足十星期充當軍師兼形象指導。Greta有廚藝天份，看媽媽示範一次便會跟着做，2歲搓麵糰還能想像，但4歲怎樣煮意粉呢？「她站在凳上炒，我在旁邊不讓她掉下來。」Greta媽說：「比賽前的四星期，我們買了二十多本世界各地的烹飪書試菜，然後選可能會考的賽題！」入圍可靠天才，勝出還得靠努力，「Tip題」不是練精學懶，而是有備而戰。

因為爸爸說明年她生日會帶她到越南旅行，Greta現正努力鑽研越南菜。
問到將來是否會專攻烹飪科，Greta說：「我想上大學會選商科，因為我想有足夠的知識去開餐廳。」廚神不是煉成的，是興趣裁成的。

酒樓似皇宮

眼前的小廚神Greta是一位亭亭玉立的少女，第一次來港的她非常興奮，短短兩天便去了山頂、廟街，甚至半島High tea。「香港的點心很好吃！款式比澳洲多很多，酒樓好像皇宮！跟唐人街細小舊式的完全不同。」來到購物天堂，自然要去Shopping，「衣服嘛，只買了滑雪褸給爸爸，我和媽媽去逛超巿，有太多調味料和食材，一定要買回去試試！」興奮得似乎在想如何用瑤柱做沙律了。

泰式紅咖喱大蝦

蝦是Greta爸爸最拿手處理的食材，汁料內加入小蝦的做法很少見，原來是她爸爸想人客吃汁的時候也吃到蝦肉的心思！咖喱汁有小廚神最愛的芫荽和青檸，因為香味濃郁，且可以整棵整個使用。Greta憑這道菜輕易過了大廚Luke Nguyen那關，得到偶像賞識，令她特別難忘！

份量：3人　時間：20分鐘　難度：3

材料：
大蝦3隻、香茅1條、泰國檸檬葉2片、青檸（取皮）1個、薑（去皮切粒）約10克、紅椒半個、辣椒1/2隻、蒜頭1瓣、芫荽1束、魚露半湯匙、茄醬半茶匙、豉油1湯匙、麻油1茶匙、小蝦（出水去殼）6隻、荷蘭豆100克、橄欖油少許、椰奶150毫升、水適量、飯1碗、芫荽葉（切碎）100克

做法：
1. 香茅切去頭部，撕去表層，用刀面拍壓香茅令它出味，然後切段，跟檸檬葉、青檸皮、薑、紅椒、辣椒、蒜頭和芫荽同放入攪拌機略打，加入魚露、茄醬、豉油和麻油打勻成咖喱醬料。

2. 熱鑊加橄欖油，放大蝦每面煎30秒，加入3湯匙醬料煎至金黃，撈起放入預熱150℃焗爐焗5分鐘。

3. 油鑊放入小蝦和荷蘭豆快炒，加入醬料炒勻，落椰奶拌煮。如咖喱變得乾身，可加入適量水份，煮至自己理想稠度，便可淋在白飯面。

4. 把焗好的大蝦放咖喱汁上，芫荽葉碎完成。

山核桃鱸魚配紅薯

這也是Greta參賽作品之一，原名「山核桃鱸魚，紅薯蘋果水牛芝士芝麻菜」。當年比賽場地是美國迪士尼樂園，參加者要玩尋寶遊戲，找到甚麼煮甚麼，這道菜的材料是她左挖右掘配搭出來的！她最愛把魚沾脆漿的過程，因喜歡做菜雙手很messy的感覺。我問Greta，「砌碟」的動作在家會做嗎？「當然會！我很喜歡把菜式放得好看才上菜，這是我的手工！」果然是個真正要講究賣相的廚師。

份量：２人　時間：40分鐘　難度：3

材料：
紅薯底：小紅薯（去皮切粒）2個、火箭菜1束、韭菜（切碎）/蘋果（切碎）各80克、水牛芝士（切碎）40克、鹽/黑胡椒碎/橄欖油/牛油各少許

山核桃鱸魚：鱸魚80克、山核桃（切碎）/牛油（加熱煮溶）/麵粉各60克、鹽/黑胡椒碎各1/4茶匙、麵包糠240克、雞蛋（拂勻）1隻、羅勒（切碎）/橄欖油各少許

做法：
1. 小紅薯加鹽、黑胡椒碎和橄欖油調味，放入預熱150℃焗爐焗20分鐘至全熟，壓爛成薯茸。

2. 滾水下煮蘋果20分鐘至軟身，隔乾水份後壓成蘋果茸。

3. 熱鑊下橄欖油和牛油炒韭菜至軟身。

4. 紅薯茸加入蘋果茸、水牛芝士、韭菜和火箭菜拌勻成紅薯底，舀入蛋糕杯倒轉覆在碟上。

5. 麵粉加入鹽和黑椒拌勻，麵包糠加入山核桃和牛油拌勻，鱸魚依次沾上調好味的麵粉、雞蛋和山核桃麵包糠。

6. 熱鑊加橄欖油煎鱸魚至兩面金黃，放入預熱150℃焗爐焗10分鐘至香脆，斜放在紅薯上，撒上羅勒碎，完成。
