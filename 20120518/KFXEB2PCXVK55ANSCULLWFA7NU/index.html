Soler樂與怒　火辣煮意

意大利人天生就有烹調天份，正如擁有一半意籍血統的Soler，是圈中公認的煮得型男。兩兄弟最近就為新開的潮人Cafe──TonnoKitchen設計了多款意粉菜式，仲會失驚無神穿起廚師服做客席大廚，親自「煮理」給客人品嚐，有型男下廚，小心未吃已被迷暈！

記者：趙珮倩
攝影：伍慶泉
場地：TonnoKitchen（31253888）
灣仔杜老誌道5號1樓

AlDente煮少兩分鐘

講起意粉，曾定居意大利、樂於下廚的Soler敢認是專家，長短幼粗扁圓通心的，他們瞭如指掌：「煮通心的意粉，汁料可配稀一點，讓更多汁液流入中間；長意粉適合配上稠一點的汁料，方便掛汁。烚意粉也有秘訣，想煮得AlDente（彈牙），通常可按包裝上的烚煮時間減兩分鐘；還有，烚煮意粉的水記得留用，待意粉回鑊炒時加入適量的烚煮水，有埋芡的作用，可令汁料更易煮至稠身。」說得似模似樣，立即看看他們煮成如何？

香辣海鮮扁意粉

這道傳統意菜叫SeafoodArrabbiata，意大利人一般會以蒜頭、西班牙香腸和辣椒來炒意粉，原來Arrabbiata意大利文的意思是憤怒，所以一定要夠辣味！

材料

扁意粉200克、三文魚柳/斑塊各40克、墨魚仔2隻、新鮮虎蝦/新鮮帶子各1隻

配料：洋葱碎/蒜茸/乾葱碎/鯷魚碎/水瓜柳/西班牙腸粒/法國香葱碎各1茶匙、指天椒（切碎）/乾辣椒（切碎）各1隻

調味料：海鹽/黑胡椒碎各適量、番茄沙沙醬1湯匙

做法

1. 扁意粉烚3-4分鐘至剛熟，瀝水備用。

2. 煎香三文魚柳、斑塊、墨魚仔、虎蝦及帶子，撒海鹽和黑胡椒碎調味，盛起備用。

3. 燒熱鑊下油，先爆香洋葱碎、蒜茸、乾葱碎、西班牙腸碎略炒，再下其餘配料炒香。

4. 下扁意粉略炒，將海鮮回鑊，落番茄沙沙醬，撒黑胡椒碎兜勻即可。

Tonno吞拿魚扁意粉

餐廳名叫Tonno，意大利文的意思是吞拿魚，所以Soler就設計了這款吞拿魚扁意粉，簡單得來具心思。

材料

吞拿魚（刺身用）120克、扁意粉200克

配料：洋葱碎1湯匙、蒜茸/乾葱碎/鯷魚碎/水瓜柳各1茶匙、黑橄欖/車厘茄各4粒

調味料：檸檬黑椒粉/黑胡椒碎/法國香葱碎各適量

做法

1. 將扁意粉烚約3-4分鐘至剛熟，瀝水。

2. 燒熱鑊下油，以中火爆香洋葱碎、乾葱碎、鯷魚碎、蒜茸、黑橄欖、水瓜柳、車厘茄。

3. 加入（1）炒勻，撒入檸檬黑椒粉、黑胡椒碎、法國香葱碎兜勻上碟。

4. 將吞拿魚略燒表面，切片，放在扁意粉上即成。

紅酒燒牛臉頰肉意大利粉

牛臉頰肉燜得十分軟腍，配上香濃的紅酒汁料伴意粉同吃，滋味無比。

材料

牛臉頰肉160克、意大利粉（已烚）200克

汁料：洋葱碎/紅蘿蔔碎各30克、西芹碎20克、黑胡椒/丁香粒各少許、紅酒125毫升、番茄膏1湯匙、水適量、麪粉1茶匙

配料：西班牙香腸粒/蒜茸/法國香葱碎各1茶匙、洋葱碎1湯匙

做法

1. 先煎香牛臉頰肉表面至金黃。

2. 爆香洋葱碎、紅蘿蔔碎、西芹碎，下牛臉頰肉炒勻。

3. 加入紅酒，下黑胡椒、丁香、番茄膏拌勻，加水浸過面，滾起後收細火，燜2小時至腍身，加入麪粉拌勻備用。

4. 另起鑊爆香西班牙香腸粒、蒜茸、洋葱碎，加意大利粉炒勻，下法國香葱兜勻，即可伴以紅酒汁牛臉頰上碟。
