私家廚房：麥曦茵夠薑摩天輪

Boxing Day除了約埋一班朋友拆禮物，其實一起弄大型的甜點，也是十分有趣的活動！以著名編劇麥曦茵為首，創意十足的Dumb Youth就以製作薑餅來一次大搞作，只是幾位年輕人度橋演戲唱歌厲害，烹飪技巧就so far so good，惟有求教於Foodgears的陳偉民師傅。薑餅甜點，吃了暖身，製成後亦充滿成功感，啱晒呢班熱血青年！
記者：張文蕙　攝影：黃子偉

薑餅的歷史可以追溯至公元八世紀的英國，主要材料包括薑粉，味道甜中帶辣很特別。早期英國的基督徒會以各種造型的薑餅慶祝聖誕，據說第一個薑餅人來自伊利沙伯一世時期的英國宮廷，主要是給貴賓享用的。到了十九世紀，每逢聖誕節人們都愛在樹上掛上各種形狀的薑餅。此外，德國人亦很愛吃薑餅，紐倫堡更有「薑餅之都」之稱，據說因那裏聚集了不少雕刻家、工匠、畫家等，由他們親手炮製的薑餅，都美麗得如藝術品。

逃出烤箱的小人

經典的薑餅人故事相傳來自英國民間。話說一對賣麵包為生的年老夫婦，膝下無兒，一天他們烤了個小人模樣的薑餅，烤熟後這個小人竟從烤箱中跳出來，然後便展開一段追逐的情節，包括老先生、老婆婆、一隻乳牛、一隻馬兒、三個農夫都加入這場浩浩蕩蕩的追餅行列，但是薑餅小人跑得非常快，沒有人能追得上，直到他遇到一隻狡猾的狐狸，狐狸不像前頭那些人及動物，早早便洩露了想吃薑餅的念頭，只是示好的說：「無論你跑到哪裏，我都會跟着你！」薑餅小人於是對他不設防，跑到河邊時，狐狸虛情假意要帶小人渡河，然後便一步步引誘薑餅小人由尾巴、背上一路轉移到狐狸的肩膀、頭頂、鼻尖，最後咕嚕一口把薑餅人吞食了。

陳偉民師傅@Foodgears

曾任職於多家五星酒店，亦曾於香港大型連鎖烘焙公司出任生產總監。經常為媒體撰寫特色菜單，與大眾分享特式甜點。

薑餅摩天輪

在陳師傅指導下，這個薑餅摩天輪，由梁曉豐（Anjo）、黃溢濠（Yat Ho）、岑珈其（Kaki）合力完成。三個大男孩像砌模型般，越玩越起勁，雖然過程中少不免有點手忙腳亂，但花了約一小時，加上糖霜漿糊的威力，終於完成了這個有趣的製作！

份量：8人　時間：4小時　難度：★★★★★

材料：
糖430克、牛奶180克、麵粉1370克、小蘇打粉12克、人造牛油430克、葡萄糖510克、肉桂粉20克、丁香粉12克、薑粉12克、糖霜粉150克、蛋白30克、醋5毫升

做法：
1. 將糖霜粉、蛋白、醋混和成用作黏貼的「漿糊」。

2. 將糖、牛奶、麵粉、小蘇打粉、人造牛油、葡萄糖、肉桂粉、丁香粉、薑粉混和成麵糰。

3. 冷藏2小時，壓成3mm厚。

4. 以180℃焗製25分鐘。

5. 壓出大圓的餅模，於中心點壓出小圓。用不同的圖案壓出其他圖案，貼在大圓上。

6. 切割不同的形狀，砌出心目中的薑餅，事前最好先預備圖樣。

7. 以（1）將薑餅黏貼好即成。

薑餅玻璃曲奇

最愛與陳師傅研究弄甜品心得的阿Lu當日剛巧在場，見Dumb Youth學弄薑餅甜品這麼過癮，也技癢即場就為我們示範她拿手的薑餅玻璃曲奇！她說時間許可的話，甚至可將薑餅以液態糖霜黏一個中空的3D薑餅人，中間放入小型的燈泡，定成為派對的亮點所在！

份量：5人　時間：3小時　難度：★★★

材料：
植物牛油420克、糖430克、糖霜（液態）510克、小蘇打粉12克、肉桂碎20g、丁香碎12克、生薑碎12克、Foxy糖15粒、牛奶180毫升、麵粉1,370克

做法：
1. 將植物牛油、糖、小蘇打粉、牛奶、麵粉，加入肉桂碎、丁香碎、生薑碎，混合成麵糰，冷藏2小時。

2.取出麵糰壓成3厘米厚。

3. 裁成大薑餅人圖案，掏空要做成像玻璃的位置。

4. 以圓形模具印出麵糰，以不同形狀的模具掏空中心。

5. 將（3）、（4）之材料以180℃焗25分鐘。

6. Foxy糖敲碎，放入焗好的薑餅的掏空位置，並以180℃焗製。

7. 以糖霜畫上薑餅人樣子及衣飾。

8. 在底板上掃上糖霜，將薑餅立於糖霜層上即成。

小貼示：

避免使用過於粉碎的Foxy糖塊，易令「玻璃」起泡，有礙通透度。

巨型薑味馬卡龍

馬卡龍總令人難以抗拒，如果弄一個巨型的更能成為派對亮點！麥曦茵話平日很少煮嘢食，接受這個挑戰的條件就是要帶兩個好幫手：林耀聲及王敏奕（Venus），聽說Venus除了演戲外，烹調食物都好拿手！今日梁師傅就教他們以薑味的忌廉，去弄一個超大型的馬卡龍，製作時聞到香噴噴Venus已不停偷食，難為生手的阿聲沒停過手，唧cream時初時有點難看，但誠意十足，後期進步了不少！麥曦茵希望團體都能有這種精神，以自己的力量將創作盡力做好，她說：「我們不敢奢望改變世界，世界正無可避免地改變我們；但每一個人只要抱着想改變的決心，我們也可以改變世界，哪怕只是一點點。」

份量：5人　時間：1小時30分鐘（不計忌廉雪藏時間）　難度：★★★★

材料：
馬卡龍餅底2塊（a）、可可海綿蛋糕1塊（c）、薑味忌廉（b）/薑味餅乾/糖霜各適量、糖霜公仔數個

做法：
1.將馬卡龍餅底置底，放上可可海綿蛋糕。

2.切碎薑味餅塊，灑於蛋糕面。

3.唧上薑味忌廉。

4. 切碎薑味餅乾，灑在忌廉上，在上面唧出忌廉花紋。

5. 蓋上馬卡龍餅底，灑上糖霜，放上糖霜公仔即成。

（a）馬卡龍餅：蛋白164克、糖428克、水145毫升、杏仁粉394克、糖霜394g

做法：
1. 將糖加水，加熱至115℃。

2. 蛋白打起，加入（1）。

3. 攪拌及冷卻約10分鐘。

4. 加入杏仁粉及糖霜於（3）後放入唧袋，唧出9吋大的圓形。室溫放15分鐘。

5. 以155℃焗16分鐘，之後冷藏備用。

（b）薑味忌廉：黑朱古力200克、忌廉1,000克、糖60克、薑粉4克、丁香粉2克、肉豆蔻粉2克、肉桂粉2克

做法：
1. 薑粉、丁香粉、肉豆蔻粉、肉桂粉混合後過篩。

2. 500克忌廉、糖及（1）以85℃加熱。

3. 將朱古力加入（2），持續拌勻。

4. 冷卻至50℃，將餘下忌廉加入，冷凍12小時。

5. 用時以機器打起忌廉。

（c）可可海綿蛋糕：可可粉26克、麵粉25克、生粉25克、糖120克、蛋黃125克、牛油清60克、蛋白115克

做法：
1. 將麵粉過篩。

2. 蛋黃及糖混合，打成淡黃色。

3. 將（1）及（2）、牛油清、生粉、糖、蛋白、可可粉混合。

4. 倒入圓模後以210℃焗約15分鐘。

飲杯蛋酒　輕鬆吓

蛋酒是聖誕節的特色飲品，由雞蛋、牛奶、忌廉和朗姆酒調製而成，源於英國，但在美加更為流行，就連奧巴馬面對美國財政懸崖，也輕鬆說先喝杯蛋酒，有信心把事情搞好；看來蛋酒的確是美國人鬆弛神經的妙方，今天就教大家三種簡單的做法。
記者：容慧心

蛋酒Latte

材料：
白蘭地30毫升、蛋利口酒60毫升、牛奶90毫升、Expresso 1 Shot、蜜糖20毫升、肉桂粉半茶匙、肉桂條2條

做法：
白蘭地、蛋利口酒、牛奶、Expresso和蜜糖加入調酒杯內搖勻，倒到酒杯內，撒上肉桂粉，最後加肉桂條作裝飾，完成。

朱古力蛋酒

材料：
白蘭地30毫升、蛋利口酒60毫升、牛奶90毫升、蜜糖20毫升、朱古力粉1茶匙、朱古力漿適量

做法：
1.朱古力漿放入細唧嘴，唧在杯內作裝飾。

2.白蘭地、蛋利口酒、牛奶、蜜糖和朱古力粉加入調酒杯內搖勻，倒到已裝飾的杯內，完成。

薑味蛋酒

材料：
白蘭地30毫升、蛋利口酒60毫升、牛奶90毫升、薑汁30毫升、蜜糖20毫升

做法：
白蘭地、蛋利口酒、牛奶、薑汁和蜜糖加入調酒杯內搖勻，倒到酒杯內，完成。
