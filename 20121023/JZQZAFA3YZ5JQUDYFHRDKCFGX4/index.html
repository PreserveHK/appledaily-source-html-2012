百合　滾一分鐘搞掂

記得小時候吃的百合都是從藥房買回來一片片的乾百合，媽媽用它來煲湯或糖水，後來街巿出現了一個個像蒜頭的鮮百合，說是直接炒吃的日常蔬菜，媽媽就買回去試試，我那時不愛它的苦味，但現在覺得味道蠻吸引的，可能由於人長大了，對吃苦也另有番體會吧！　記者：容慧心　攝影：楊錦文　示範：Norma

乾和濕兩種百合各有好壞，乾百合是黃黃一片片的百合，質地很硬，食用時準備功夫較多，煮之前要用水浸泡至軟身才容易煮透，但味道較甜，質地亦較粉；鮮百合顏色偏白，食用較方便，洗淨後去頭尾便能烹調。不過鮮百合味道帶苦，保存時間也不及乾百合長，選購時要挑肉白且肥厚的，如非立即食用，應放陰凉處。
百合含有蛋白質、脂肪、鉀、食物纖維、維他命E和維他命C等，主要有清熱潤肺、益氣止咳、利便和寧神的功效。但要注意，若有風寒咳嗽、便溏、腎臟功能不佳者則不宜食用。

百人續命野草根

百合名字的由來有三種說法，最簡單的就是因為它根球部份由「百」（很多）瓣「合」起來組成，因以名之。又有說百合能治療坐立不安、心神恍惚等，遂取「百合病」之意稱之。還有一個來自有趣的傳說，話說從前中國東面海域小島有一群海盜，他們經常搶劫沿海的漁村，有一次海盜把糧食、女人和小孩從漁村劫回島上，但幾天後再出海時遇上大風暴，結果全部遇難；被虜的村民糧食開始短缺，惟有找到甚麼都試着吃，他們找到一種像大蒜的野草根，發現吃起來又甜又飽肚，連有肺病的人吃了也漸漸好轉。一年過去，一群採草藥的人無意中來到島上，於是將村民送回漁村，並帶走讓村民續命的「大蒜」回去栽培，發現它對止咳與鎮靜神經果然有效！後來，人們便以島上「合起來有一百人受惠」這個意思命名了「百合」。三種說法有兩個跟百合的食用價值有關，可見中國人有多推崇百合的補健功效。

鮮百合海鮮豆腐羹

羹，自古以來便是中國人的高尚食品，從造字來看，「羹」是用羊等肉類煮成的美食，主要為富有人家享用，平民百姓則用蔬菜煮成。現在羹的配搭千變萬化，廣東人愛吃海鮮，這次就用百合加入海鮮豆腐羹，令羹有更多不同口感，拌飯也非常不錯！

材料

蝦/帶子各150克、豆腐280克、鮮百合80克、紅蘿蔔（切碎）/青豆各60克、蛋白1隻、雞湯500毫升、水250毫升、鹽/胡椒粉少許、生粉水少許、薑片/紹興酒適量

做法

1.
水中放薑片和紹興酒煲滾，放入蝦和帶子出水，撈起後切成小粒備用。

2.
鮮百合切去頭尾啡色部份，逐瓣拆開；豆腐切成小塊備用。

3.
湯鍋放入雞湯和水煮滾，放入海鮮、豆腐、青豆和紅蘿蔔碎，滾起後轉慢火煮約15分鐘，最後加入百合煮約1分鐘。

4.
蛋白快速拌成蛋花加入，然後落生粉水拌勻，最後加鹽和胡椒粉調味，完成。

南瓜蒸鮮百合

媽媽最喜歡煮一道被我稱為「澱粉王」的菜，就是南瓜芋頭煮栗子！澱粉和澱粉同煮永遠好吃，但一次過吃太多碳水化合物還是有點膩吧；鮮百合也含澱粉質，但質地爽脆，且帶有甜味，配南瓜同煮，吃起來也不會太膩滯。

材料

南瓜400克、鮮百合1個、水100毫升、生粉2茶匙、雞粉1茶匙、鹽少許

做法

1.
南瓜洗淨去籽，切2cm厚片，鋪放蒸碟上。

2.
鮮百合切去頭尾啡色部份，逐瓣拆開，撒在南瓜片上。

3.
水滾，大火蒸南瓜百合15分鐘。

4.
水、生粉、雞粉和鹽拌成芡汁，以平底鍋煮熱，淋上蒸熟的南瓜百合面，完成。

貼士

最好選用老南瓜，食用前可撒點桂花，味道更香甜。

西芹百合炒蝦仁

西芹膳食纖維高，又能降血壓，是很好的減肥食品。鮮百合味道微帶點苦，西芹剛好中和了鮮百合這個味道。西芹、百合和蝦仁都是不需要長時間烹調的食物，煮起來快捷方便，吃起來也符合保存食物原有營養的原則。

材料

鮮蝦20克、鮮百合80克、西芹120克、紅蘿蔔60g、水50毫升、生粉/雞粉1茶匙、鹽/蒜茸/胡椒粉/紹興酒各少許、油適量

做法

1.
鮮蝦洗淨抹乾，以胡椒粉和紹興酒調味。平底鍋落油大火燒熱，炒蝦至半熟撈起備用。

2.
西芹洗淨撕去表面粗纖維，斜切成段；紅蘿蔔刨皮切片；鮮百合切去頭尾啡色部份，逐瓣拆開，備用。

3.
燒熱平底鍋，落油爆香蒜茸，加2湯匙清水炒勻西芹、紅蘿蔔、鮮百合，加入蝦仁拌炒約3分鐘。

4.
最後落水、生粉、雞粉和鹽拌成的芡汁，炒勻後上碟，完成。

鮮百合綠豆糖水

百合蓮子紅豆沙，相信大家都吃過吧！其實，百合煲綠豆沙也一樣可口。綠豆除了消暑，當中所含卵白質和磷脂還可增進食欲，所含的多糖又可降血脂和降低小腸對膽固醇的吸收，以之做糖水是非常保健的甜品。

材料

綠豆200克、薏米100克、鮮百合20克、冰糖3塊、水2公升

做法

1.綠豆和薏米分別放冷水中浸泡1小時。

2.煲滾水，放入綠豆煮滾後落薏米，煮滾後轉慢火煮約45分鐘至稠身，加入冰糖調味。

3.鮮百合切去頭尾啡色部份，逐瓣拆開，加入糖水略煮即可。

貼士

糖水稠度按個人喜歡而定，喜歡稠的可煮更長時間。新鮮百合處理較方便，但喜歡用乾百合的便要提前一晚浸泡，浸至軟身後撕去表面薄膜便可烹調。
