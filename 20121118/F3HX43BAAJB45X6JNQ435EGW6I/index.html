我做煮：晨起樂活

身兼不同角色，要應付多種範疇工作，特別是寫作、做設計時，心亂就甚麼也做不了。常跟同樣忙碌的朋友說，要忙而不亂，要擁有片刻完全屬於自己的時光，早起一點享受美好早餐，先呵護自己再工作，一天會過得暢快些。大清早，由身心疲乏至朝氣勃勃，是新的一天，有新的機會。在早晨最清醒靈巧的時段，與其用來放在惱人工作上，不如先幹些自己喜歡的事，學習另一專長。日本有許多不同的晨早學習班，為希望進修的人而設，說不定他們可因此學得一技之長，依自己的喜好討生活。

喜歡早起，享受晨早獨處的寧靜時刻，先做個早餐，泡杯茶，再看書，安安靜靜地展開新的一天。整天下來，心情也特別祥和安樂。不論從營養或生活角度看，早餐也是重要一餐，為自己弄一頓豐富早餐，為家人預備晨早美點，是滋養心靈的好法子。總要感到飽滿豐盛才開始一天的工作，早餐吃不飽，整天就是昏昏欲睡，做甚麼也提不起勁。
友人說早已給工作折騰透，還要早起做早餐實在是大懲罰。我卻是不管多忙多累，也要預備早點，愛吃早點的習慣要感謝爸爸。他愛早起，我總像他，睡多了總覺迷迷糊糊。小學中學時爸爸會為我預備早點，有時只是簡簡單單的腿蛋三文治、即冲好立克或牛奶，又或是火腿通粉，爸爸也會在天未亮透就到麪包店為我買菠蘿包、吞拿魚包、豬仔包，每天我總是吃飽才上學去，看同學邊跑邊趕路邊吃着才剛買的麪包，在搖晃晃的車廂中吃，我卻是早已吃飽滿足得很，笑瞇瞇地看書溫習。

一期一會的幸福

早餐的美食，可以是朱古力奶跟鬆餅。做鬆餅很簡單，先在早一晚量好材料，第二天清晨，把材料拌勻就可送進焗爐，稍待一會，廚房香氣瀰漫，熱烘烘的鬆餅烤好了。不做鬆餅，可選擇班戟伴楓蜜、通粉伴忌廉湯、英式鬆餅伴煙肉荷包蛋、三文魚青瓜三文治伴自煮奶茶， 或是中式的稀飯伴自磨豆奶。食物用喜歡的餐具盛好，吃早餐時，捧讀心愛書本，播些輕快音樂，一小時半小時也好，已可樂而忘憂。家人如愛賴床就把三文治鬆餅用餐巾包好，讓他們拿着上班上學去，好讓他們不時想到家。
旅行時，每晚就寢前也好期待明早的早餐時刻。非常喜歡Room Service的早餐。印象最深刻的是在傳統的京都旅館 「俵屋」，在低調古雅的房間中，早起先在庭園旁邊享受旅館早預備好的飲料，再由女將把早餐食物送到房間。在俵屋進餐，是一次對京都料理之美的體會。享受京都料理時，我想起「一期一會」，是一生只有一次的精神，是主人對客人的誠意，客人因而會更珍惜，細味感受。當然，在英國鄉村小鎮間的小旅館，女主人親自下廚，自烤麪包伴私房果醬加些炒蛋青豆是完全另一種氛圍，卻仍是非常幸福的時刻。

伯爵茶蛋糕

牛油蛋糕即使是前一天弄好，隔天仍美味，不用趕不用忙也可享受Homemade cake！

材料：
低筋麪粉100克、全麥麪粉20克、泡打粉半茶匙、無鹽牛油（室溫放軟）120克、黃糖50克、幼砂糖40克、雞蛋（拂勻）2隻、伯爵茶包茶葉2個、橙皮碎1茶匙、熱水2湯匙

做法：
1. 焗爐預熱至190℃。鬆餅焗盤放焗杯備用。

2. 麪粉、泡打粉篩勻，備用。

3. 取2湯匙熱水泡1個茶包約3分鐘，取出茶汁。另一茶包取出紅茶碎，備用。

4. 將軟牛油、幼砂糖及黃糖用打蛋器高速打約5分鐘成忌廉狀，分多次慢慢加入蛋汁拌勻。

5. 加入茶汁、橙皮碎拌勻。

6. 拌入紅茶葉碎，分2次加入粉料輕輕拌勻。

7. 倒入焗杯中，焗約12-15分鐘至熟，即可享用。

貼士：
要將蛋逐少逐少慢慢加入，麪粉不可以高速攪拌，否則會起筋而蛋糕就變得不鬆軟了。

芝士煙肉粒貝殼粉

芝士通粉或許是你我的童年回憶。加入不同的蔬果令營養更豐富，是早餐，也可以是Brunch。

材料：
貝殼粉100克、牛油1湯匙、蒜片1湯匙、意大利煙肉粒4湯匙、粟米粒3湯匙、西蘭花2小棵、鮮奶/淡忌廉各1/4杯、即磨巴馬臣芝士碎3湯匙、水/鹽/胡椒各適量

做法：
1. 水煮沸，加少許鹽，下貝殼粉煮至口感剛好，瀝水。

2. 西蘭花切碎。

3. 燒熱牛油，用中慢火把煙肉粒炒香，再下蒜片炒香。

4. 加入鮮奶、淡忌廉煮沸，下西蘭花、粟米粒煮一會。加入貝殼粉煮至吸收汁料。

5. 最後下芝士碎，以鹽、胡椒調味才享用。

玫瑰焦糖奶茶

晨起用鍋子煮奶茶，是我每天好生活不可或缺的儀式。時光好像慢下來，對於我可安撫心靈。

材料：
砂糖2湯匙、鮮奶1杯、水1杯、台灣紅茶葉10克、玫瑰花3粒

做法：
1. 砂糖中火煮至成焦糖，拌入少許鮮奶拌勻。

2. 加入其餘鮮奶煮熱成焦糖鮮奶。

3. 水煮沸，加入紅茶葉、玫瑰花，慢火煮半分鐘，離火泡3分鐘。隔出茶葉。

4. 加入焦糖鮮奶，即可享用。
