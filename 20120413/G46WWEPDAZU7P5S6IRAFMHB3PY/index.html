海景蝦餃皇

蝦餃是最受歡迎的傳統點心之一，除了餡料中的蝦肉要新鮮彈牙，那塊晶瑩又煙韌的蝦餃皮還要搓得薄，而且蝦餃皮摺法還有規定，要摺出12至14摺才算合格，用包蝦餃來考手藝便可見功力了。

材料

蝦餃皮：澄麪/生粉各320克、水320毫升

餡料：中蝦肉400克、肥豬肉60克

調味料：鹽1/3茶匙、雞粉/糖各半茶匙、生粉/生油各1茶匙、胡椒粉/麻油各少許

做法

1. 煲滾水，把熱水倒入澄麪，拌成熟澄麪，再分批搓入生粉成蝦餃皮。

2. 取15克蝦餃皮，並用刀壓成薄片。

3. 將中蝦肉、肥豬肉切成幼粒，加調味料拌勻至起膠成餡料。把皮放在手心，曲起，包入約25克餡料。

4. 將皮摺叠成14摺為蝦餃形狀，把蝦餃放蒸籠中，以大火蒸5分鐘即成。

貼士：把手心曲起，蝦餃的大小跟曲起的位置剛好，便是合適的大小。

蟹皇帶子燒賣

點心孖寶永遠不會少了燒賣的份兒，別以為燒賣只是用燒賣皮包住肉餡，做法很容易，其實要把餡料包好後保持挺身結實，蒸出的燒賣才不會散開兼好吃。

材料

瘦肉200克、中蝦肉300克、肥豬肉60克、帶子8隻、燒賣皮16塊、蟹黃/生粉各適量

調味料：鹽/雞粉各半茶匙、糖/生粉各1茶匙、胡椒粉/麻油各少許

做法

1. 蝦肉去腸，洗淨切成幼粒。瘦肉、肥豬肉一同切成幼粒備用。

2. 中蝦肉、肥豬肉、瘦肉向同一方向拌勻至起膠，加調味料拌勻成餡。拿起一塊燒賣皮，放入餡料，用手包成棋子般，握實。

3. 帶子上下片開，灑入少許生粉拌勻，放在燒賣頂上，拍實。

4. 在燒賣頂再放少許蟹黃作裝飾，將燒賣放入蒸籠，以大火蒸約8至10分鐘即可。

貼士：把餡料包入皮後，要用力握實，蒸好才不會鬆散。

狀元煎茶粿

平日見的茶粿多是蒸的，海景軒的茶粿卻是先蒸後煎。鹹香的餡料包在帶甜味的茶粿皮內，要包出這個橄欖形的茶粿，便要注意皮的厚薄，兼要留意形狀，才可做出精緻效果。

材料

茶粿皮：糯米粉500克、熟澄麪320克、粘米粉/砂糖各120克、豬油220克、水220毫升、杞子30粒、韭菜30條、鹽少許

餡料：豬肉/雞肉各120克、菜脯100克、花生/蝦米/冬菇（已浸）/唐芹各45克、韭菜65克

調味料：鹽/雞粉各半茶匙、糖/生粉各1茶匙、胡椒粉/麻油各少許

做法

1. 豬肉、雞肉、菜脯、蝦米、冬菇、韭菜、唐芹分別切成幼粒。

2.熱鑊下油，落豬肉、雞肉、菜脯、冬菇、花生、蝦米爆香，加調味料拌勻。

3.煲滾水，下少許鹽、韭菜、唐芹汆水，撈起隔水，拌入（2）中成餡料。

4. 糯米粉、粘米粉、砂糖、水搓勻成粉糰，再加入熟澄麪和豬油搓成皮料，用保鮮紙包好。

5. 取20克皮料壓成塊，皮邊要揑成碗狀，包入15克餡料做成茶粿形狀，收口。

6. 在茶粿面繞上韭菜，放上杞子，隔水以大火蒸8分鐘，再放入加了少許油的熱鑊中，煎香底部即成。
