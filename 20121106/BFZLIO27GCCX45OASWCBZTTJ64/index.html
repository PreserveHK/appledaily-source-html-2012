內線王：市場風險降　基金掟房託 - 范董

港股連日造好，投資者的風險胃納開始提升，但防守性甚強的房託基金一樣升個不停，連番創新高。
不過，與此同時，有個別基金已靜悄悄減持房託基金，由此看來，若港股升勢持續的話，「價高、息低」的房託基金遲早被投資者嫌棄，由防守性變成失去防守性。

回報低欠防守性

據聯交所資料顯示，瑞士資產管理公司Veritas Asset Management AG旗下的The Real Return Group，於10月30日減持置富產業信託（778）133.5萬個基金單位，每單位平均價6.014元，持有量由5%，降至4.92%的毋須披露水平。
這家The Real Return Group，顧名思義，一向喜歡投資實際回報較高的公司，另外亦持有高息公路股浙江滬杭甬（576）。
這家資產管理公司或許有見置富產業價格升至現水平，息率回報降至不足4.2厘，可謂食之無味，所以才願意高位割愛。
至於其他房託基金，以2011年計算，領匯（823）的息率回報更只有3.3厘，與首20隻股息率最高的藍籌股相比，領匯的息率更低，如此情況之下，作為以高息作賣點的房託基金，何來再有長線持有的吸引力？
