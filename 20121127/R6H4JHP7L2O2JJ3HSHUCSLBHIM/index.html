內地客蝕40萬沽The Java

【持貨兩年】
樓市前景未明，有內地客先行沽貨離場。英皇集團（163）旗下的北角單幢樓The Java，一名來自溫州的業主，剛以1950萬元沽出單位，持貨兩年，扣除使費蝕約40萬元。
樓市交投淡靜，部份對樓市態度欠樂觀人士先沽貨。名釆策略營業董事姚雲龍表示，整體樓市依舊未見起色，僅剩下個別零星單幢物業成交。北角The Java高層B室，面積1205方呎，樓齡僅一年，剛獲一名區內用家斥資以1950萬元購入，呎價1.618萬元。
據知，該名沽貨業主乃溫州客，於2010年10月斥資1898.4萬元向發展商一手購入，持貨兩年，雖然賬面微賺51.6萬元，但扣除釐印費及佣金等使費，明賺實蝕約40萬元離場。

爵士花園六年漲近倍

另外，香港置業營業經理容靖宇表示，何文田豪宅爵士花園3座低層C室，面積1323方呎，剛連車位以1440萬元售出，呎價1.088萬元。原業主於06年中以730萬元買入，賺710萬元，升值約97%。
