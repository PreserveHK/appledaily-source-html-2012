越跌越買
債王無懼危機低吸意西債

PIMCO創辦人兼聯席投資總監格羅斯雖然早已斷言美國聯儲局會推出第三輪量寬（QE3），但資料顯示他在當月繼續減持美國國債，同時亦無懼危機，買入了水深火熱的西班牙和意大利國債。

續減持美國債

人稱債王的格羅斯，在9月份已是連續第三個月減持美債，因為他擔心美國面臨的財政問題及極寬鬆貨幣政策將削弱投資者對美債的信心。據PIMCO網站周三公佈的數據顯示，截至9月底於旗下總回報基金（Total Return Fund）資產組合中美國國債所佔比例已降至20%，低於8月底的21%及7月時的33%，不過仍是該債券基金投資的第二大資產，但當中已沒有10年以上的長債。與此同時，格羅斯增持了非美元發達國債券，上述債券在基金中的比例從8月底的7%上升至11%。至於總回報基金中持倉比例最大的仍是按揭抵押證券（MBS），9月底在組合中所佔的比例為49%。雖然獲得債王格羅斯追捧，但意大利昨日拍賣37.5億歐元3年期國債，息率仍上升至7月中以來的最高水平。今次出售規模達到拍賣目標上限，息率為2.86厘。而上月拍賣同類債券時，息率為2.75厘。
