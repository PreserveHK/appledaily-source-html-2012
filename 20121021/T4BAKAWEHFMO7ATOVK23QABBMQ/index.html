我做煮：養身隨心

身邊朋友看我外表總以為我不吃肉不愛辣不飲咖啡怕膩怕甜。誰知我從來也不是那種偏吃、計算卡路里、每天吃大堆維他命丸的人。一直相信健康來自心境明澄開朗，每天在有限的時間條件中尋找令自己身邊人舒服的生活方式是其一；遠離不喜歡不相投的人事物是其二；追隨生活理想投入熱愛工作是其三。至於吃，與其花心神看營養標籤，與其皺眉說不要甚麼不吃甚麼 ，我倒寧願花時間行市場找好食材，用心煮慢慢吃，在烹調享受美味間尋快樂。大部份食材也沒有絕對的好沒有絕對的壞，吃甚麼跟做甚麼一樣還在於適可而止在於平衡。不要把自己弄得太緊張，不要非吃非不吃不可。聽自己的身體心靈說話，自己最清楚需要甚麼。
因喜歡讀飲食散文，有關養身、健康飲食法也在閱讀間學到些。好像是黃双如出的《双如談食》，透過回憶、小故事介紹好食材的效用、烹調方法，閒時看一兩篇是好窩心的。又好像最近重讀的《禪食慢味》，作者是曾經擔任禪寺食堂大廚十多年的宗哲和尚，在山庵開設精進料理烹飪班「禪味會」，書中以說故事般的語調，訴說一道道如天地恩賜的精進料理。全書沒有宗教的說教味，只有對料理對食材的深刻體悟，其中的道理，其實也是我健康的烹調之道。在這裏跟大家分享一些。　攝影：劉永發

尊重天然味道

好像對於我所喜歡的重視食物原味，作者說：「濃厚調味會抹煞了食材本身的原味。為了讓食材本身所擁有的味道發揮，只能是淡味。讓本身所擁有的味道活出生命力。」而淡味，「不只是薄味而已，必須讓食材本身所擁有的味道展現出生命力。」選食材，要不時不食，注重『旬』，即是「指生命的新鮮感、潤澤度與年輕。」烹調時，「要考慮如何用最好的烹調法與心情去料理，本着一顆體貼食材的心去烹飪，最後懷着感恩的心情享用。」在有關長壽一章中，作者說：「善盡天地資源的食材，不偏愛的加以調理。不正是所謂的『飲食中有中道』嗎？」　　喜歡烹飪以來，自己預備日常三餐時，是越煮越簡約，越吃越簡單。不喜歡繁複的烹調方法，一點肉加時令蔬菜、養生食材，盡量享受食物的原味真味，重點在於不同口感滋味的食材交互搭配掌握其中平衡之道。
尊重季節，尊重食材天然的味道，愛物惜物，是我喜歡的養生方式。如此淡泊簡單卻淡雅綿長，這也是我嚮往的生活風格吧。

紅棗雲耳蒸雞

好喜歡吃雞!特別愛加入養生食材木耳、紅棗、陳皮、杞子，吃過後元氣十足!

材料：

光雞 （去頭尾後斬件）半隻、花菇（浸軟切粗絲）2隻、薑（切絲）3片、木耳（浸軟）2片、紅棗（去核）5粒、陳皮（浸水去瓤切絲）1片

調味：頭抽1湯匙、紹酒1茶匙、生粉1/2茶匙、糖/鹽各1/4茶匙、芝麻油/胡椒/鹽各1/8茶匙

做法：

1.
光雞斬件、花菇浸軟切絲、薑切絲、紅棗去核切碎備用。

2.
把調味料拌勻，加入雞件、花菇、木耳，醃半小時。

3.
燒熱油，下薑絲爆香，加入雞件、花菇、雲耳炒一會。

4.
大火燒開水，放入雞件，撒上紅棗、陳皮，蒸15-20分鐘至熟即成。

牛肝菌黑豚杞子鍋飯

豚肉是正氣的肉食，加入燕麥飯中，伴乾菌雜菜，一鍋飯已有足夠的營養!

材料：

蒜片1湯匙、黑豚肉片8片、牛肝菌10克、冬菇3隻、甘笋50克、燕麥米50克、日本米100克、杞子2湯匙、木魚高湯/麻油各適量、日本醬油、味醂各1　1/2茶匙

做法：

1.
米洗淨，待30分鐘。瀝水。

2.
冬菇、牛肝菌浸軟切絲。取浸水。甘笋切絲。

3.
米、燕麥米放飯鍋，加入牛肝菌浸水及木魚高湯共約180毫升（水量按不同的米而定）。

4.
燒熱2茶匙油，加入蒜片炒香，再牛肝菌、冬菇，下日本醬油、味醂，炒2分鐘，加入甘笋絲炒拌，與杞子下鍋中。中火煮沸，轉中火煮10分鐘。

5.
黑豚肉片撒些鹽後加入，再煮5分鐘至飯熟，離火焗10分鐘，下少許麻油拌勻享用。

南北杏川貝梨茶

秋天天氣乾燥，呼吸系統較弱易咳嗽不舒服。秋天，是梨的季節，甜美的梨生津潤燥，最宜秋天享用。加入潤肺治咳的南北杏、川貝，是清潤的甜品。

材料：

梨1個、南北杏20克、川貝10克、冰糖適量

做法：

1.
雪梨連皮洗淨，剖半，去中心部份。

2.
川貝略搗碎。

3.
雪梨中放入南北杏、川貝、冰糖，放燉盅，隔水燉1小時30分鐘，即可享用。
