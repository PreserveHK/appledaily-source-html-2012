濃甜台南虱目魚　魚蛋攀龍成國宴

台灣美食小吃除了大雞排這種炸物，其實還有很多清淡又健康的美食，像出名是小吃重地的台南，當地人最喜歡吃虱目魚，無論熬粥或做魚丸，統統都鮮美得讓人難忘。今次由來自台南的美食專家楊老師，親自示範幾款她曾做過的虱目魚國宴料理，口味更地道。

記者：謝翠玲
攝影：陳盛臣
示範：台南周氏蝦捲國宴廳經理楊淑燕

虱目魚全身寶

來自台南周氏蝦捲的國宴廳經理楊淑燕除了「掌煮」國宴，本身也是美食專家和烹飪老師，「虱目魚除了風味佳，還很有益，台南人會以虱目魚來補身。」這款魚是一種遍佈於全世界熱帶與亞熱帶水域的魚類，比較怕冷，夏天才最肥。而牠含有豐富骨膠原、17種氨基酸，而其DHA和EPA比鰻魚更高，對降低膽固醇和預防血酸、提升腦部功能也有助益，在當地有狀元魚之美譽。

日劏千條虱目魚

周氏蝦捲每天需要用過千條虱目魚，專門負責劏魚的許正德師傅，曾創下日劏千條虱目魚的紀錄，可說工多藝熟，對於如何把魚不同部份分割可說毫無難度。虱目魚的每個部份用途和質感也不一樣，魚肚滑嫩、魚背口感像雞肉、魚皮膠質豐富、內臟則甘香，所以要分割清楚來分開用途。

﹝四招分身﹞

「玩轉全臺灣——寶島好味道」

想品嚐周氏蝦捲的原店風味？不一定親身到台南，台灣觀光協會和東港城將於6月16至17日，在東港城一樓中庭便有一場台灣地道美食展，除可品嚐虱目魚外，亦有其他台灣美食。

虱目魚丸魚皮湯

此湯加油條是台南人很地道的一種吃法。虱目魚丸跟我們平日吃的潮州魚蛋質感不同，魚味濃，質感實淨，平日店中是以大量的虱目魚骨來熬湯底，魚味鮮甜不過很大陣仗。家庭版改用豬骨湯底來提升魚的鮮味一樣夾。

材料

虱目魚丸300克、虱目魚皮200克、豬骨上湯11/2公升、芹菜碎/娃娃菜（切碎）各100克

調味料：雞粉/鹽/白胡椒/麻油各適量

做法

1. 煲滾豬骨上湯，加入虱目魚丸煲至滾起，加入娃娃菜灼熟，下雞粉和鹽調味。

2. 魚皮切小片，放入湯中燙兩分鐘，即可盛起。

3. 最後下芹菜碎、白胡椒和麻油同吃即可。

貼士：

魚皮不要燙太久，免得會變韌，失去爽滑口感。

安平虱目魚肚粥

台灣人的粥一般是稀飯，但這道粥則以生米加上湯熬成，是台南人至愛的早餐之一，一般配以油條同吃。店中是以魚骨熬湯底，家庭版改以上湯，配虱目魚肉和蠔仔一樣充滿鮮味。

材料

虱目魚肚1片、虱目魚肉/蠔仔各200克、笋絲100克、白米300克、上湯2公升、芹菜/油條/蒜頭酥各適量

調味料：雞粉/鹽/胡椒粉各適量

做法

1. 白米洗淨，加入上湯，一同煮滾後，收細火煲一小時成粥底。

2. 虱目魚肚切小塊；魚肉切小丁。

3. 將笋絲、蠔仔、虱目魚肚和魚丁加入（1）中，大火滾五分鐘，加雞粉和鹽調味。

4. 把粥盛碗中，灑上蒜頭酥、芹菜和胡椒粉，最後加入油條同吃。

貼士：

虱目魚不要過早加入粥中，以免破壞魚肉本身的滑嫩口感。

香煎虱目魚肚

虱目魚肚嫩滑肥美，魚脂甘香豐富，以最簡單的香煎做法便可表現出魚肚位的優點。

材料

虱目魚肚1片、鹽/椒鹽粉各少許、檸檬（切角）半個

做法

1. 虱目魚肚去掉魚骨，兩面抹少許鹽略醃。

2. 平底鑊中加少許油，燒熱，先把魚肉那面向鑊底，以大火煎至定形，收細火繼續煎。

3. 當魚肉邊位離鑊，反轉煎魚皮那面，轉大火煎香至表面金黃，上碟。

4. 灑上椒鹽粉，並放上檸檬角，吃時唧上檸汁伴食即可。

貼士：

怕煎虱目魚肚易散開，可在臨煎前在魚身抹少許地瓜粉或生粉。
