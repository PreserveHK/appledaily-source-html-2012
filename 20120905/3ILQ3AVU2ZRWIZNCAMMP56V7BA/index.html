蘋機會：獨家技．防失真

現時大部份影音系統都是使用HDMI進行影像及聲音的傳輸，不過，隨住玩家們的要求越來越高，開始注意傳輸時出現的Jitter誤差。而Denon最近推出首部使用Denon Link HD傳輸技術的DBT-3313UD藍光機，號稱Jitter Free，聲音會更有細節、更立體！　記者：許福合　攝影：陳盛臣　模特兒：Shanna@PureModels

已經是第二代的Denon Link HD，機背會多了一個Denon Link HD專用插頭，不過在接駁上其實不會難倒一般玩家，只需以同軸線接駁，同時HDMI線亦要接好，就能在設定內打開Denon Link HD。筆者搵來D&M市場經理洪國柱解構這個新技術：「利用Denon Link HD，AV擴音機會將時鐘訊號輸出到藍光碟機，兩機會使用共同時鐘訊號，在輸送數碼音頻時，Jitter誤差可以大幅減少，主要可以改善音樂的細節及立體感。」不過用家要留意這是「獨家」技術，其他品牌機種未能享用，而且暫時亦只有Denon剛推出的DBT-3313UD藍光機及AVR-3313擴音機支援。
專售影音器材的金富視聽科技，影音專家阿強亦表示高檔的影音器材上，唔少客人都會追求器材的一致性：「增加了Denon Link HD後，聲音的層次感明顯上升，尤其立體感更豐富，串流音樂音色更通透。」加上DBT-3313UD可以將視頻及音頻，分別以兩組HDMI，獨立輸出到電視及擴音機，更可保證到互相不受干擾，對聲畫質素提升都有幫助。
DBT-3313UD藍光機除了有新傳輸技術外，讀碟精準及靜音亦是賣點，而且啟動快速，在備用狀態只需2.5秒就可啟動，其他網絡功能亦相當齊全，能支援24bit/192kHz WAV及FLAC音樂檔。 手拿iPhone或Android手機， 安裝好《Denon Remote App》後，就可以控制晒藍光機及擴音機，簡單又方便。
